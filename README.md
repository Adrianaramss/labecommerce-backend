# Projeto Labecommerce

  Projeto do back-end onde é aplicado toda a base de criação de uma API vinculada a um banco de dados real.

# Contéudo abordados
- NodeJS

- Typescript

- Express

- SQL e SQLite

- Knex

- Postman
